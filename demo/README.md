# Demo Targets

This folder contains the self-contained system that SpecForge AI analyzes, tests, and reports on.

## Contents

- `specs/`: product requirements, acceptance criteria, and OpenAPI contract
- `api-go/`: Go service under test with intentionally seeded API defects
- `mutator-rust/`: Rust-based boundary and adversarial case generator
- `contracts/`: Solidity smart-contract slice with Foundry tests and Hardhat smoke flow
